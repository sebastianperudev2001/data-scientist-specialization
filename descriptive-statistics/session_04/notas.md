# Coeficiente de correlacion

- R = Covarianza / (Desviacion estandar de X \* Desviacion estandar de Y)
- R = 1, correlacion positiva perfecta
- R = -1, correlacion negativa perfecta
- R = 0, no hay correlacion

En terminos sencillos, mide si 2 variables se mueven juntas.

Si mi coeficiente de relacion es 1, no existe.
